# CleanTrack

A waste management mobile app that helps residents track garbage trucks, report issues, and keep their neighborhoods clean.

## Features

- **Live Truck Tracker** — See where the garbage truck is in your neighborhood in real time with estimated arrival times.
- **Report an Issue** — Report illegal dumping, bin damage, or missed collections with photo evidence and track their status.
- **Recent Activities** — View a feed of recent reports and updates from your neighborhood.
- **Segregation Guide** — Learn how to properly sort waste with a searchable, color-coded guide.

## Getting Started

### Prerequisites

- Node.js v18+
- Expo CLI
- Android Studio or Xcode

### Installation

```bash
git clone https://github.com/your-username/cleantrack.git
cd cleantrack
npm install
npm start
```

### Running on Device

```bash
npm run android
npm run ios
```

## Tech Stack

- **Mobile** — React Native / Expo
- **Maps** — Google Maps API
- **Backend** — Node.js + Express
- **Database** — Firebase
- **Auth** — Firebase Auth
